## Общая оценка готовности

**≈ 85 %**

Основной интерфейс, навигация, триггеры, мониторинг, DBC-интеграция и графики сигналов реализованы и запускаются. Остаётся доработать полноценную интеграцию DBC в гибкую логику, реализовать полноценный CAN-шлюз и провести тестирование на реальных устройствах всех платформах. Сборки под Windows, macOS и Linux подготовлены, но требуют запуска в GitHub Actions для финальной проверки артефактов.

## Ключевые библиотеки

- **cantools** — основной парсер DBC и декодирование физических значений.
- **carbus-lib** (импортируется как `carbus_async`) — заложена для будущей работы с CAN/ISO-TP/UDS транспортом; для парсинга DBC не используется.
- **pyqtgraph** — графики сигналов в реальном времени.
- **PySide6** — GUI-фреймворк.
- **pyserial** — работа с COM-портами.
- **matplotlib** — оставлен для совместимости с существующим графиком `can_graph_tab.py`.
