## Ключевые библиотеки

- **cantools** — основной парсер DBC и декодирование физических значений.
- **carbus-lib** (импортируется как `carbus_async`) — заложена для будущей работы с CAN/ISO-TP/UDS транспортом; для парсинга DBC не используется.
- **pyqtgraph** — графики сигналов в реальном времени.
- **PySide6** — GUI-фреймворк.
- **pyserial** — работа с COM-портами.
- **matplotlib** — оставлен для совместимости с существующим графиком `can_graph_tab.py`.
